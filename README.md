# 🎓 Students Table – React CRUD Application

## 📌 Project Overview
This project is a **Students Management Table** built using React.  
It allows users to **add, edit, delete, search, and manage student records** in a dynamic table interface.

All CRUD operations are handled on the **frontend using React state and LocalStorage**, meaning no backend server is required.

The application also supports **Excel file download, pagination, and loading simulation**, providing a professional user experience.

---

## 🚀 Features

- Display Students Table with columns:
  - Name
  - Email
  - Age
  - Actions (Edit/Delete)

- Add Student with Form Validation
- Edit Student with Pre-filled Data
- Delete Student with Confirmation Dialog
- Loading Spinner Simulation
- Search Students
- Pagination for Student Records
- Download Student Data as Excel File
- Professional UI using Bootstrap
- Data persistence using LocalStorage
- Fully frontend-based CRUD operations

---

## 🛠 Technologies Used

- React.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Sreekumar1024/students-table.git
```

### Navigate to project folder

```bash
cd students-table
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:5173
```
